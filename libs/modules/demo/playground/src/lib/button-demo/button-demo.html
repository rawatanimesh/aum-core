Disabled States

All button types and configurations support the [disabled]="true" state. The visual treatment and pointer events are suppressed consistently across variants.

Text only

Tooltip

Pass [tooltip] to show a Material tooltip on hover. Commonly used on icon-only buttons to expose the action label, but available on all types.

Custom Width

Set [width] to any valid CSS length to stretch the button — useful for form submit buttons or full-width CTAs in constrained containers.
