Default Size (24×24)

No inputs needed. Default is 24×24 matching Angular Material — scales automatically with the global UI scale setting via rem().

[width] overrides both width and height (square icon). Value is in px and is converted to rem with --ui-scale applied, so it still scales.

Color Inheritance (currentColor)

Default behavior — icons inherit the text color of their parent. No [color] input needed. Use this for nav items, buttons, and any context where the parent already sets the right color.
