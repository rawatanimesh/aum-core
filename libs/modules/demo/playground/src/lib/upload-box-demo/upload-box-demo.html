Upload Box Component

A comprehensive file upload component with drag-and-drop, validation, progress tracking, and forms integration.

1. Reactive Forms with Auto-Upload

Upload documents with automatic upload on selection. Max 5 files, 5MB each. Integrated with Angular Reactive Forms.

2. Standalone Mode with Manual Upload

Select multiple files and upload manually. Max 10 files. Uses two-way data binding with [(data)].

3. Single File Mode (Profile Picture)

Upload a single image file with preview. Max 2MB. Perfect for profile pictures and avatars.

4. Custom Validation & Manual Upload

Select files with custom validation (no "temp" in filename), manually upload them, then submit the form once all uploads are complete.
