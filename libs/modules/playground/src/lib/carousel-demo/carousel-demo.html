5. Infinite Scroll (Default Speed)

Carousel with [infiniteScroll]="true" - automatically scrolls continuously. Default animation duration is 20 seconds.

👆 Hover to pause the animation (20s duration)

6. Infinite Scroll (Custom Speed)

Control animation speed using [animationDuration]="10". This carousel scrolls twice as fast (10 seconds).

👆 Faster animation (10s duration)

7. Scroll by Item (Default)

Navigation scrolls one item at a time. This is the default scrollMode="item".

8. Scroll by Viewport

Navigation scrolls by visible width with scrollMode="viewport". Allows scrolling multiple items at once.

API Reference

Inputs

Input Type Default Description
carouselItems CarouselItem[] [] Array of items to display
showArrows boolean true Show/hide navigation arrows
scrollMode 'item' | 'viewport' 'item' Scroll behavior: 'item' scrolls one item at a time, 'viewport' scrolls by visible width
infiniteScroll boolean false Enable automatic infinite scrolling animation
animationDuration number 20 Duration in seconds for infinite scroll animation (only applies when infiniteScroll is true)
itemHeight string 'calc(25rem * var(--ui-scale))' Height of carousel items
itemWidth string 'calc(25rem * var(--ui-scale))' Width of carousel items

Outputs

Output Event Type Description
cardClick CarouselCardClickEvent Emitted when a card is clicked
